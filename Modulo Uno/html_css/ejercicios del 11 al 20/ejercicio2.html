SELECTORES

En esta section trataremos de los tipos de selectores. Hay 4 tipos e selectores y son:

  • Selector universal
  • selector de elemento
  • Selector de clase
  • Selector de IDE

Selector universal "*"

El selector universal es el que cubre todo el cuerpo de nuestra pagina web y se llama usando el asterizco " * "



Selector de elemento " h2 "

El selector de elemento es a quel que los cambios se dan en solo el elemento llamado y se llama con el nombre del elemento, como por ejemplo (h2).



Selector de clase

El selector de clase es aquel elemento que con tiene una clase dentro de un elemento o un contenedor o div y se usa para no llamar a un elemento y tambien para aplicar la tecnica de flex box y la de padre e hijo. ejemplo (ul class="lista")



Selector de ID

El selector de ID no puede haber dos ID con el mismo nombre no es como la clase el nombre la clase si se puede repetir varias veces el Id no solo una vez tiene que ser unico. (ejemplo: section id="contenedor")
